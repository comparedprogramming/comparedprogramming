# Hello World in C

## Compiling instructions for Debian based distributions

Install [GCC](https://gcc.gnu.org)(GNU Compiler Collection):

```bash
$ sudo apt update
$ sudo apt install build-essential
```

Compile the source code:

```bash
gcc helloworld.c -o helloworld
```

Run the binary:

```bash
./helloworld
```
